Instructions Notes
1. Select the Calculator tab and select your hospital from the drop down menu. Certain default input parameters will now prepopulate.
2. Fill in as many orange input variables as you have data for. These will customize the calculation to your hospital's cost infrastructure.
3. Check green output cells.
